# pqR 2019-02-19 on Debian

There is already a docker image for this combination:

- [Source](https://github.com/nuest/pqr-docker)
- [Docker Image](https://hub.docker.com/r/nuest/pqr)

## Usage

Pull docker image:

`make pull dockeRs`

Run container:

`make run dockeRs`

Run shell on the container:

`make shell dockeRs`
